ROMA VOICE ASSISTANT
Roma is a python based voice assistant that can perform various task using voice commands.

FEATURES:
voice command
Text to Speech responses
Tell current time
Tell current date
Open google
Open youtube
Open calculator
open command prompt
Open VS code
Open Gmail
Play songs on youtube
Wikipedia search
Memory save and recall
Joke generator

TECHNOLOGIES USED:
Python
SpeecRecognition
pyttsx3
Pywhatkit
Wikipedia
Pyjokes

HOW TO RUN:
Install Python
Install required libraries
Run roma.py
